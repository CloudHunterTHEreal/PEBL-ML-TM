PEBL ML TM
Классификация текстовых документов с использованием Вероятностной тематической модели, которая учитывает неизвестные слова, рассчитывая вероятность отнесения документа к новой теме. В задаче обработки потока текстовых документов вероятность отнесения документа к новой теме, позволит отследить необходимость переобучения модели, или расчета тем для новых встреченных в новых документах слов. Предложенный подход позволяет обучить многозначный классификатор текстовых документов, в качестве обучения в котором используются примеры интересующего класса.
